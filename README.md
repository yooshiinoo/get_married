# get married

## サイト概要
### サイトテーマ
入籍後のやることリストを自分流にカスタマイズ出来るサイト

### テーマを選んだ理由
自分自身が行う事となり、なれない手続きが多かったので、あれば便利だと考え、このテーマにしました。

### ターゲットユーザ
入籍にて届出が必要な人

### 主な利用シーン
入籍前準備をする時、入籍後様々な手続きを行う時。
